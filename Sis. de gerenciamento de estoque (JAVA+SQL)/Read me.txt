📦 Sistema de Cadastro e Controle de Produtos com Java, MySQL e Power BI
🔍 Descrição
Este projeto consiste em um sistema simples, mas funcional, de cadastro e remoção de produtos, desenvolvido em Java com conexão a um banco de dados MySQL, e complementado com um dashboard interativo no Power BI para visualização dos dados.

🚀 Funcionalidades
✅ Cadastro de produtos (nome, unidade, preço, data)

✅ Remoção de produtos por ID

✅ Armazenamento das informações em um banco MySQL

✅ Integração com Power BI para visualização dos dados em tempo real

✅ Interface de terminal orientada a menus

🛠️ Tecnologias utilizadas
Tecnologia	                                        Função
Java	                               Lógica do sistema e conexão com banco
MySQL	                               Armazenamento dos dados
JDBC	                               Driver de conexão entre Java e MySQL
Power BI	                           Visualização gráfica dos dados
SQL	                                   Criação e manipulação das tabelas

🧪 Como executar:

Crie um banco chamado sis_estoque
Execute a SQL com a tabela ESTOQUEL (com campos como id_produto, tipo_produto, unidade_produto, etc.)

Baixe a pasta "Codigo" e arraste para o visual studio code
Em Conexão.JAVA coloque a URL, USER e PASSWORD do seu banco de dados (Caso tenha mudado a porta, usuario ou a senha)
logo apos vá em "App.JAVA"

Abra o Power BI e conecte o relario com o MySQL
Faça a coneção normalmente seguindo os passos 

📊 Dashboard Power BI
O dashboard exibe métricas como:

📦 Total de produtos cadastrados

📈 Média de preços

🕒 Produtos por data de cadastro

📊 Distribuição por tipo de produto

💡 Ideal para empresas que desejam controlar estoque com visualização em tempo real.

📁 Base de dados
Exemplo de tabela:

CREATE TABLE ESTOQUEL (
    id_produto INT PRIMARY KEY AUTO_INCREMENT,
    tipo_produto VARCHAR(100),
    unidade_produto INT,
    preco_unidade DOUBLE,
    data_cadastro VARCHAR(10)
);
