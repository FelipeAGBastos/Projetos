👤 Sistema de Cadastramento de Usuários em Java com MySQL

🔍 Descrição
Este projeto consiste em um sistema de terminal simples, porém funcional, para cadastrar e remover usuários em um banco de dados MySQL, utilizando a linguagem Java e a biblioteca JDBC para realizar a conexão.

🚀 Funcionalidades
✅ Cadastro de usuários (nome, CPF, telefone)

✅ Armazenamento em banco de dados relacional (MySQL)

✅ Interface de terminal com menus de operação

🛠️ Tecnologias utilizadas
Tecnologia	Função
Java	Lógica da aplicação
MySQL	Armazenamento dos dados dos usuários
JDBC	Conector Java ↔ MySQL
SQL	Criação e manipulação das tabelas

🧱 Estrutura do Projeto
📁 src/
 ┣ 📁 DAO/
 ┃ ┗ UsuarioDAO.java
 ┣ 📁 entity/

 ┣ 📁 conexao/
 ┃ ┗ Conexao.java
 ┣ App.java


🧪 Como executar

baixe o arquivo .sql e abra no MySQL ou:

Configure o banco de dados:
Crie um banco chamado usuarios_cad
Execute a SQL com a tabela USUARIO_CAD

CREATE TABLE USUARIO_CAD (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nome VARCHAR(100),
    cpf VARCHAR(14),
    tel VARCHAR(20)
);

Baixe o arquivo no Visual Studio Code ou outros e em "Codigo Cad" vá em "Conexão.JAVA" e Coloque as informaçoes de URL, USER E PASSWWORD (caso tenha mudado)
Vá em "APP.java" e alteres os parametros desejados para o registro que deseja em seu banco de dados e rode o codigo.


🧠 Aprendizados
Conexão de sistemas Java com banco de dados via JDBC

Criação de DAO (Data Access Object) para isolar regras de banco

Utilização de estrutura de menus e leitura via Scanner

Tratamento de exceções com try/catch

Modelagem de entidade Usuario com encapsulamento (get/set)
