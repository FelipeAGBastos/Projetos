👤 Sistema de Cadastramento de Usuários em Java com MySQL
🔍 Descrição
Este projeto consiste em um sistema de terminal simples, porém funcional, para cadastrar e remover usuários em um banco de dados MySQL, utilizando a linguagem Java e a biblioteca JDBC para realizar a conexão.

🚀 Funcionalidades
✅ Cadastro de usuários (nome, CPF, telefone)

✅ Remoção de usuários por ID

✅ Armazenamento em banco de dados relacional (MySQL)

✅ Interface de terminal com menus de operação

🛠️ Tecnologias utilizadas
Tecnologia	Função
Java	Lógica da aplicação
MySQL	Armazenamento dos dados dos usuários
JDBC	Conector Java ↔ MySQL
SQL	Criação e manipulação das tabelas
