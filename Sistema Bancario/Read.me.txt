Simulador de Sistema Bancário Completo

Descrição
Projeto educacional desenvolvido com foco no setor financeiro, simula as principais operações de um sistema bancário utilizando Java e MySQL. A aplicação permite o gerenciamento completo de usuários e contas, com funcionalidades como cadastro, movimentações financeiras, pagamentos e controle de saldo.

Este projeto foi pensado para demonstrar conhecimentos técnicos aplicados à realidade de instituições financeiras, com ênfase em boas práticas de programação, organização de dados e integração com banco de dados.


Funcionalidades
Cadastro de usuários e contas bancárias

Operações de depósito e saque com validação de saldo

Pagamento de contas vinculado a contas específicas

Consulta de saldo com exibição de dados em tempo real

Validação de login com autenticação simples

Interface em linha de comando (CLI), com fluxo de navegação intuitivo



Tecnologias Utilizadas
Java (JDK 17+)

MySQL (banco de dados relacional)

JDBC (Java Database Connectivity)

Padrão DAO (Data Access Object)

Paradigma de Programação Orientada a Objetos

Git e GitHub 




Como Executar
Pré-requisitos
Java JDK instalado
MySQL instalado e configurado
Conector JDBC MySQL disponível no classpath
IDE para execução (VS Code, IntelliJ, Eclipse etc.)


Passos
Clone o repositório:

Crie o banco de dados sistema_bancario no MySQL

Execute o script SQL disponível na pasta Dados MySQL

Compile e execute o projeto a partir do arquivo App.java

Acesse o menu principal e navegue pelas funcionalidades
