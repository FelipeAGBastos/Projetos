# 🤖 Robo

## 📖 Descrição
Este programa em Java simula um desafio interativo com um robô antigo que testa a bateria do usuário 🔋 e sua habilidade para seguir comandos em um labirinto 🧩. O usuário deve fornecer a porcentagem da bateria e digitar uma sequência correta de comandos e distâncias para completar o desafio.

## ⚙️ Como funciona
- O programa inicia com uma introdução textual simulando um diálogo com um robô 🤖.
- O usuário deve informar a carga da bateria em porcentagem (%).
- Caso a bateria seja insuficiente (menor ou igual a 80%), o programa indica que não é possível prosseguir ⚠️.
- Se a bateria for suficiente, o usuário deve inserir 7 comandos na sequência correta, cada um com uma direção e uma distância.
- Comandos válidos são:
  1. ➡️ Direita, 4 passos
  2. ⬇️ Baixo, 9 passos
  3. ➡️ Direita, 6 passos
  4. ⬇️ Baixo, 7 passos
  5. ➡️ Direita, 6 passos
  6. ⬆️ Cima, 16 passos
  7. ⬅️ Esquerda, 9 passos
- Se algum comando for errado, o jogo termina com uma mensagem de ❌ "Game over".
- Ao completar com sucesso, o programa exibe uma mensagem de 🎉 parabéns e o “prêmio” é o aprendizado e conhecimento de Java 💻.

## 🚀 Como executar
javac robocop_v3_final.java
java robocop_v3_final
Siga as instruções no console, fornecendo as entradas solicitadas.
